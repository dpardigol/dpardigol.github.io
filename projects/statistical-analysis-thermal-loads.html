Project Overview

This project focuses on analyzing thermal loads in primary loop piping systems in nuclear power plants. The goal was to apply data-driven techniques to model thermal stresses, detect patterns in operational data, and predict maximal stress locations using machine learning methods.

Key Contributions

  • Parametrization of thermal loads based on operational data from sensors and historical records.
  • Clustering of thermal load patterns to identify recurring operating regimes using unsupervised learning techniques.
  • Prediction of maximal stress using supervised learning models, helping to anticipate potential failure zones.
  • Visualization of thermal patterns and stress predictions for engineers using advanced plotting techniques.
  • Development of Python scripts for automating preprocessing and analysis pipelines.

Technical Details

Data Processing

Raw operational data from nuclear piping systems was cleaned and normalized. Features such as temperature, pressure, and flow rates were extracted and prepared for analysis.

Clustering Analysis

Clustering algorithms such as K-Means and DBSCAN were applied to identify patterns in thermal load profiles. This enabled the classification of operational regimes for better understanding of stress variations.

Stress Prediction

Supervised learning models including Random Forest and Gradient Boosting were trained to predict maximal stress points based on operational conditions. Model performance was validated using cross-validation techniques.

Project Outcomes

  • Efficient classification of thermal load patterns, improving predictive maintenance strategies.
  • Enhanced understanding of stress variations under different operating conditions.
  • Automated data analysis scripts reduced manual processing time by 50%.
  • Deliverables contributed to internal knowledge base at Framatome for nuclear piping analysis.

Learnings & Skills Developed

  • Practical application of unsupervised and supervised machine learning in an industrial context.
  • Handling large-scale operational datasets and performing feature engineering.
  • Experience in creating visualizations for engineering decision support.
  • Improved coding skills in Python for data analysis and automation.
